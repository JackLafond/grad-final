Context:

    Visualizing Draft Data: Hive Plot

    Scroll to see how it works.

    The hive plot is made up of three axes with each axis representing a certain category. The top left axis represents colleges, the right axis represents the NFL teams, and the bottom left axis represents a players the rank in the NFL.

    A node along the top left axis represent the different college conferences. A node on the right axis represents each of the 32 NFL teams. A node on the bottom left axis represents the rank of a player, so a player with a lot of accolades will be closer to the bottom.

    A line between two nodes signifies a relationship, and the connection of a line throughout the hive plot marks a player.

    Now that you understand the plot go back up and interact with the plot. Change the rank of accolades, the draft year, but most importantly, hover over the nodes on the accolades axis to see which players rank the highest or lowest.
